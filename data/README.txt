Data files used by exmples

Note file with extnesion .raw are raw data file with no headers and are suitable
for importing into ImageJ with user supplied size and type.  These file are efficiently
read and written by the cxbinio routines.

ives512.png    : greyscale 512 x 512 image of boats in St Ives harbour as png
ives512.raw    : same image as raw 8-bit data
ives512rgb.raw : colour version of image  512 x 512 in rgb raw format.
ivesBIG.jpg    : original 5184 x 3456 image from which the others were cropped.
